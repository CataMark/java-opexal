General observations:

  • dialog windows can be closed also by hitting Esc without any effect on application data
  • after a period of 30 minutes of inactivity your session will expire and you'll be asked to sign in
  • on session expiry the page might become unresponsive; accessing the page from the left side menu or refreshing it should solve this issue
  • the timestamp in table records is according with UTC (Coordinated Universal Time)

Language selection

Changing the language (from upper-right corner of the first page) enables inserting values for allocation keys or planning documents with region specific characters for thousands grouping and decimal separator (helpful when pasting from Excel):

  • the number format for Romanian language is: 100.000,00 (dot for thousands grouping and comma for decimal separator)
  • the number format for English language is: 100,000.00 (comma for thousands grouping and dot for decimal separator)
